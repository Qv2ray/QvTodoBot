# QvTodoBot

QvTodoBot is a Telegram bot designed for the Qv2ray project.

## Commands

- `/start`: I love Qv2ray!
- `/todo <content>`: add to-do item
- `/gettodo`: get to-do list
- `/toggle <number>`: toggle to-do item status
- `/remove <number>`: remove to-do item
